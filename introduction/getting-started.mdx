## Pré-requisitos

Antes de começar, certifique-se de ter:

<Check>Uma conta ativa na plataforma</Check>
<Check>Chave de API gerada (consulte [Autenticação](/authentication/api-keys))</Check>
<Check>Conhecimento básico de requisições HTTP</Check>

## Configuração Rápida

<Steps>
  <Step title="Obtenha suas credenciais">
    Acesse o painel de controle e gere suas chaves de API na seção de configurações.
    
    <Warning>
      Mantenha suas chaves de API em segurança. Nunca as compartilhe publicamente ou as inclua em repositórios de código.
    </Warning>
  </Step>

  <Step title="Faça sua primeira requisição">
    Use o exemplo abaixo para testar a conectividade com a API:

    <CodeGroup>
      ```bash cURL
      curl --request GET \
        --url https://api.example.com/v1/status \
        --header 'Authorization: Bearer YOUR_API_KEY'
      ```

      ```javascript JavaScript
      const response = await fetch('https://api.example.com/v1/status', {
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY'
        }
      });
      const data = await response.json();
      console.log(data);
      ```

      ```python Python
      import requests

      headers = {
          'Authorization': 'Bearer YOUR_API_KEY'
      }

      response = requests.get('https://api.example.com/v1/status', headers=headers)
      print(response.json())
      ```

      ```php PHP
      <?php
      $ch = curl_init();
      curl_setopt($ch, CURLOPT_URL, 'https://api.example.com/v1/status');
      curl_setopt($ch, CURLOPT_HTTPHEADER, array(
          'Authorization: Bearer YOUR_API_KEY'
      ));
      curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
      $response = curl_exec($ch);
      curl_close($ch);
      echo $response;
      ?>
      ```
    </CodeGroup>

    <Tip>
      Substitua `YOUR_API_KEY` pela sua chave de API real.
    </Tip>
  </Step>

  <Step title="Explore os endpoints">
    Navegue pela [Referência da API](/api-reference) para descobrir todos os recursos disponíveis.
  </Step>
</Steps>
