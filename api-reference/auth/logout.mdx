## Descrição

Este endpoint permite que o usuário realize logout da plataforma, invalidando o token de acesso atual. Após o logout, o access_token não poderá mais ser utilizado para autenticação.

## Autenticação

<ParamField header="Authorization" type="string" required>
  Bearer token que será invalidado.
  
  **Formato:** `Bearer {access_token}`
</ParamField>

## Resposta de Sucesso

<ResponseField name="message" type="string">
  Mensagem confirmando que o logout foi realizado com sucesso.
</ResponseField>

## Comportamento

<AccordionGroup>
  <Accordion title="O que acontece após o logout?">
    - O access_token atual é invalidado e não pode mais ser usado
    - Todas as requisições subsequentes com esse token retornarão erro 401
    - O usuário precisa fazer login novamente para obter novos tokens
  </Accordion>

  <Accordion title="O refresh_token também é invalidado?">
    Sim, em sistemas seguros, tanto o access_token quanto o refresh_token são invalidados durante o logout para garantir que a sessão seja completamente encerrada.
  </Accordion>

  <Accordion title="Preciso fazer logout antes de fechar a aplicação?">
    Por boas práticas de segurança, sim. Sempre realize logout quando o usuário encerrar a sessão, especialmente em dispositivos compartilhados.
  </Accordion>
</AccordionGroup>

## Notas Importantes

<Note>
  Após o logout, será necessário fazer login novamente através do endpoint `/auth/login` para obter novos tokens de acesso.
</Note>

<Tip>
  Em aplicações frontend, remova os tokens armazenados (localStorage, sessionStorage, cookies) após receber confirmação do logout.
</Tip>

<Warning>
  Se você não possui mais acesso ao token (por exemplo, ele foi perdido), não é possível invalidá-lo através deste endpoint. O token expirará automaticamente após seu tempo de vida.
</Warning>

## Exemplo de Implementação Completa

```javascript
// Função de logout com limpeza de estado
async function logout() {
  try {
    const token = localStorage.getItem('access_token');
    
    const response = await fetch('https://api.monitora.com/auth/logout', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${token}`
      }
    });
    
    if (response.ok) {
      // Limpar tokens do armazenamento local
      localStorage.removeItem('access_token');
      localStorage.removeItem('refresh_token');
      
      // Redirecionar para página de login
      window.location.href = '/login';
    }
  } catch (error) {
    console.error('Erro ao fazer logout:', error);
  }
}
```
