## Descrição

Retorna uma lista paginada de todos os monitores (agentes) pertencentes ao usuário autenticado. Suporta filtros de busca e paginação para facilitar a navegação em grandes volumes de dados.

## Autenticação

<ParamField header="Authorization" type="string" required>
  Bearer token do usuário autenticado.
  
  **Formato:** `Bearer {access_token}`
</ParamField>

## Parâmetros de Query

<ParamField query="page" type="number" default="1">
  Número da página para paginação.
  
  **Exemplo:** `1`
</ParamField>

<ParamField query="limit" type="number" default="10">
  Quantidade de itens por página.
  
  **Exemplo:** `10`
  
  **Valores aceitos:** 1-100
</ParamField>

<ParamField query="search" type="string">
  Termo de busca para filtrar monitores por nome ou descrição.
  
  **Exemplo:** `campanha janeiro`
</ParamField>

<ParamField query="status" type="boolean">
  Filtrar por status do monitor (ativo/inativo).
  
  **Exemplo:** `true`
</ParamField>

## Resposta de Sucesso

<ResponseField name="data" type="array">
  Array contendo os monitores encontrados.
  
  <Expandable title="Objeto Monitor">
    <ResponseField name="id" type="string">
      ID único do monitor (UUID).
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Nome do monitor.
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Descrição do monitor.
    </ResponseField>
    
    <ResponseField name="status" type="boolean">
      Status do monitor (ativo/inativo).
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Data de criação (ISO 8601).
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      Data da última atualização.
    </ResponseField>
    
    <ResponseField name="groups_count" type="number">
      Quantidade de grupos associados.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Metadados da paginação.
  
  <Expandable title="Propriedades">
    <ResponseField name="total" type="number">
      Total de registros encontrados.
    </ResponseField>
    
    <ResponseField name="page" type="number">
      Página atual.
    </ResponseField>
    
    <ResponseField name="limit" type="number">
      Itens por página.
    </ResponseField>
    
    <ResponseField name="total_pages" type="number">
      Total de páginas disponíveis.
    </ResponseField>
  </Expandable>
</ResponseField>

## Filtros Avançados

<Tip>
  Combine múltiplos parâmetros de query para refinar sua busca:
  - Use `search` para encontrar monitores específicos
  - Use `status` para filtrar apenas monitores ativos ou inativos
  - Ajuste `limit` para controlar a quantidade de resultados por página
</Tip>

## Notas Importantes

<Info>
  Os monitores são retornados ordenados por data de criação (mais recentes primeiro).
</Info>
