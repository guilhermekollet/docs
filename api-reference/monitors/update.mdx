title: "Atualizar Monitor"
api: "PATCH /monitors/{id}"
description: "Atualiza as informações de um monitor existente."
auth: "bearer"
---

## Descrição

Este endpoint permite atualizar os dados de um monitor específico, como nome, descrição e status.

## Autenticação

<ParamField header="Authorization" type="string" required>
  Bearer token do usuário autenticado.
  
  **Formato:** `Bearer {access_token}`
</ParamField>

## Parâmetros de Path

<ParamField path="id" type="string" required>
  ID único do monitor a ser atualizado (UUID).
  
  **Exemplo:** `b0cd8450-bd1b-486e-87b3-8eeeb96436f2`
</ParamField>

## Corpo da Requisição

<ParamField body="name" type="string">
  Novo nome do monitor.
  
  **Exemplo:** `Monitor Atualizado`
</ParamField>

<ParamField body="description" type="string">
  Nova descrição do monitor.
  
  **Exemplo:** `Descrição atualizada do monitor`
</ParamField>

<ParamField body="status" type="boolean">
  Novo status do monitor (ativo/inativo).
  
  **Exemplo:** `false`
</ParamField>

## Exemplos de Código

<RequestExample>

```bash cURL
curl --request PATCH \
  --url https://api.monitora.com/monitors/b0cd8450-bd1b-486e-87b3-8eeeb96436f2 \
  --header 'Authorization: Bearer SEU_ACCESS_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Monitor Atualizado",
    "status": false
  }'
```

```javascript JavaScript
const monitorId = 'b0cd8450-bd1b-486e-87b3-8eeeb96436f2';

const response = await fetch(
  `https://api.monitora.com/monitors/${monitorId}`,
  {
    method: 'PATCH',
    headers: {
      'Authorization': 'Bearer SEU_ACCESS_TOKEN',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'Monitor Atualizado',
      status: false
    })
  }
);

const data = await response.json();
```

```python Python
import requests

monitor_id = "b0cd8450-bd1b-486e-87b3-8eeeb96436f2"
url = f"https://api.monitora.com/monitors/{monitor_id}"
headers = {
    "Authorization": "Bearer SEU_ACCESS_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "name": "Monitor Atualizado",
    "status": False
}

response = requests.patch(url, json=payload, headers=headers)
print(response.json())
```

</RequestExample>

<Note>
