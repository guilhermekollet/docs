## Descrição

Este endpoint permite verificar quantas contas do Facebook o usuário já cadastrou e qual é o limite do seu plano atual.

## Autenticação

<ParamField header="Authorization" type="string" required>
  Bearer token do usuário autenticado.
  
  **Formato:** `Bearer {access_token}`
</ParamField>

## Resposta de Sucesso

<ResponseField name="current_usage" type="number">
  Quantidade de contas atualmente cadastradas.
</ResponseField>

<ResponseField name="plan_limit" type="number">
  Limite de contas permitido pelo plano.
</ResponseField>

<ResponseField name="remaining" type="number">
  Quantidade de contas disponíveis para cadastro.
</ResponseField>

<ResponseField name="plan_name" type="string">
  Nome do plano atual.
</ResponseField>
