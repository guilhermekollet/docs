## Descrição

Este endpoint permite cadastrar uma nova conta do Facebook na plataforma, possibilitando o acesso e gerenciamento das AdAccounts (Contas de Anúncios) associadas a ela.

## Autenticação

<ParamField header="Authorization" type="string" required>
  Bearer token do usuário autenticado.
  
  **Formato:** `Bearer {access_token}`
</ParamField>

## Corpo da Requisição

<ParamField body="access_token" type="string" required>
  Access Token do Facebook obtido através do OAuth.
  
  **Exemplo:** `EAABwzLix...`
</ParamField>

<ParamField body="name" type="string" required>
  Nome identificador para a conta.
  
  **Exemplo:** `Conta Facebook - Principal`
</ParamField>

<ParamField body="user_id" type="string">
  ID do usuário do Facebook.
  
  **Exemplo:** `123456789`
</ParamField>

## Resposta de Sucesso

<ResponseField name="id" type="string">
  ID único da conta cadastrada (UUID).
</ResponseField>

<ResponseField name="name" type="string">
  Nome da conta.
</ResponseField>

<ResponseField name="status" type="string">
  Status da conta (active, inactive).
</ResponseField>

<ResponseField name="created_at" type="string">
  Data de cadastro (ISO 8601).
</ResponseField>

## Exemplos de Código

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.monitora.com/accounts/facebook \
  --header 'Authorization: Bearer SEU_ACCESS_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "access_token": "EAABwzLix...",
    "name": "Conta Facebook - Principal"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.monitora.com/accounts/facebook', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer SEU_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    access_token: 'EAABwzLix...',
    name: 'Conta Facebook - Principal'
  })
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.monitora.com/accounts/facebook"
headers = {
    "Authorization": "Bearer SEU_ACCESS_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "access_token": "EAABwzLix...",
    "name": "Conta Facebook - Principal"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```

</RequestExample>

## Notas Importantes

<Info>
  O access_token do Facebook deve ter as permissões necessárias para acessar as AdAccounts.
</Info>

<Warning>
  Verifique os limites de contas do seu plano antes de cadastrar novas contas.
</Warning>
