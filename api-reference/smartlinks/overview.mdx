## Recursos Principais

### Validação de Links

Verifique se um link de convite do WhatsApp está ativo e válido antes de usá-lo em campanhas.

### Rastreamento Avançado

Capture informações detalhadas sobre cada clique:
- Dispositivo (mobile, desktop, tablet)
- Sistema operacional
- Navegador
- Localização (IP)
- Data e hora
- User agent completo

### Paginação Inteligente

Todos os endpoints de listagem suportam paginação com metadados completos:
- Total de itens
- Total de páginas
- Indicadores de próxima/anterior
- Controle de limite por página

### Filtros Avançados

Filtre atividades por:
- Tipo (click, entry)
- Período (startDate, endDate)
- Paginação customizada

## Autenticação

<Note>
  Todos os endpoints SmartLinks requerem autenticação via Bearer token ou API token.
</Note>

**Métodos suportados:**

1. **Bearer Token** (JWT)
```bash
Authorization: Bearer seu_token_jwt
```

2. **API Token** (para consumo de créditos)
```bash
x-api-token: seu_api_token
```

## Exemplos Práticos

### Criar e Validar Link

```javascript
// 1. Validar link do WhatsApp
const validation = await fetch('https://api.example.com/v1/smartlinks/validate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    url: 'https://chat.whatsapp.com/ABC123XYZ'
  })
});

const { is_valid } = await validation.json();

if (is_valid) {
  console.log('✅ Link válido! Pode criar a campanha.');
}
```

### Monitorar Cliques

```python
import requests

# Obter atividades da campanha
response = requests.get(
    'https://api.example.com/v1/smartlinks/campaign/ABC123/activities',
    headers={'Authorization': 'Bearer YOUR_TOKEN'},
    params={
        'type': 'click',
        'page': 1,
        'limit': 50
    }
)

data = response.json()
print(f"Total de cliques: {data['meta']['total']}")
```

## Limites e Quotas

- **Rate Limit**: 100 requisições por minuto
- **Paginação máxima**: 500 itens por página
- **Retenção de dados**: 90 dias de histórico
- **Validações**: 1000 validações por dia (plano free)
