## Descrição

Este endpoint permite atualizar a imagem de perfil de um grupo do WhatsApp através da API.

## Autenticação

<ParamField header="Authorization" type="string" required>
  Bearer token do usuário autenticado.
  
  **Formato:** `Bearer {access_token}`
</ParamField>

## Parâmetros de Path

<ParamField path="groupId" type="string" required>
  ID único do grupo (UUID).
  
  **Exemplo:** `b18c6011-6b61-4e8d-b797-d0256bec296a`
</ParamField>

## Corpo da Requisição

<ParamField body="image_url" type="string" required>
  URL da nova imagem do grupo.
  
  **Exemplo:** `https://example.com/nova-imagem.jpg`
</ParamField>

<ParamField body="image_base64" type="string">
  Imagem em formato Base64 (alternativa ao image_url).
  
  **Exemplo:** `data:image/jpeg;base64,/9j/4AAQSkZJRg...`
</ParamField>

## Exemplos de Código

<RequestExample>

```bash cURL
curl --request PATCH \
  --url https://api.monitora.com/groups/b18c6011-6b61-4e8d-b797-d0256bec296a/image \
  --header 'Authorization: Bearer SEU_ACCESS_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "image_url": "https://example.com/nova-imagem.jpg"
  }'
```

```javascript JavaScript
const groupId = 'b18c6011-6b61-4e8d-b797-d0256bec296a';

const response = await fetch(
  `https://api.monitora.com/groups/${groupId}/image`,
  {
    method: 'PATCH',
    headers: {
      'Authorization': 'Bearer SEU_ACCESS_TOKEN',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      image_url: 'https://example.com/nova-imagem.jpg'
    })
  }
);

const data = await response.json();
```

```python Python
import requests

group_id = "b18c6011-6b61-4e8d-b797-d0256bec296a"
url = f"https://api.monitora.com/groups/{group_id}/image"
headers = {
    "Authorization": "Bearer SEU_ACCESS_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "image_url": "https://example.com/nova-imagem.jpg"
}

response = requests.patch(url, json=payload, headers=headers)
print(response.json())
```

</RequestExample>
