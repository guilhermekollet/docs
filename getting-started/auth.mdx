---
title: "Accounts API"
description: "A API Accounts permite gerenciar contas do Facebook e contas de anúncios (Ad Accounts) utilizadas no Monitora API."
---

## Autenticação

A maioria dos endpoints requer autenticação.

Header obrigatório:

```
Authorization: Bearer <seu_token_jwt>
```

---

## Endpoints

<AccordionGroup>

  <Accordion title="Criar Conta do Facebook" icon="plus-circle">
    Cria uma nova conta do Facebook vinculada ao usuário autenticado.

    **Método:** POST
    **URL:** `/accounts/facebook`

    <ParamField body="name" type="string" required>
      Nome identificador da conta do Facebook.
    </ParamField>

    <ParamField body="externalId" type="string" required>
      ID externo da conta no Facebook (ex: fb_123456).
    </ParamField>

    <ParamField body="accessToken" type="string" required>
      Token de acesso da conta do Facebook.
    </ParamField>

    <RequestExample>
    ```json
    {
      "name": "Conta Facebook Principal",
      "externalId": "fb_123456",
      "accessToken": "EAABsbCS1..."
    }
    ```
    </RequestExample>

    <ResponseExample>
    ```json
    {
      "id": "uuid",
      "name": "Conta Facebook Principal",
      "externalId": "fb_123456",
      "createdAt": "2026-01-10T12:00:00.000Z"
    }
    ```
    </ResponseExample>

  </Accordion>

  <Accordion title="Atualizar Conta do Facebook" icon="edit">
    Atualiza os dados de uma conta do Facebook existente.

    **Método:** PATCH
    **URL:** `/accounts/facebook/{id}`

    <ParamField path="id" type="string" required>
      ID da conta do Facebook.
    </ParamField>

    <ParamField body="name" type="string">
      Novo nome da conta.
    </ParamField>

    <RequestExample>
    ```json
    {
      "name": "Conta Facebook Atualizada"
    }
    ```
    </RequestExample>

    <Note>
      Retorna a conta atualizada.
    </Note>

  </Accordion>

  <Accordion title="Listar Contas do Facebook" icon="list">
    Lista as contas do Facebook do usuário autenticado com paginação e filtros.

    **Método:** GET
    **URL:** `/accounts/facebook`

    <ParamField query="page" type="number" default="1">
      Número da página.
    </ParamField>

    <ParamField query="limit" type="number" default="10">
      Itens por página.
    </ParamField>

    <ParamField query="search" type="string">
      Filtrar por nome da conta.
    </ParamField>

    <ResponseExample>
    ```json
    {
      "data": [
        {
          "id": "uuid",
          "name": "Conta Facebook Principal",
          "externalId": "fb_123456"
        }
      ],
      "meta": {
        "page": 1,
        "limit": 10,
        "total": 1
      }
    }
    ```
    </ResponseExample>

  </Accordion>

  <Accordion title="Criar Conta de Anúncios (Ad Account)" icon="plus">
    Cria uma nova conta de anúncios vinculada a uma conta do Facebook.

    **Método:** POST
    **URL:** `/accounts/adaccounts`

    <ParamField body="facebookAccountId" type="string" required>
      ID da conta do Facebook associada.
    </ParamField>

    <ParamField body="name" type="string" required>
      Nome da conta de anúncios.
    </ParamField>

    <ParamField body="externalId" type="string" required>
      ID externo da conta de anúncios (ex: act_123456).
    </ParamField>

    <RequestExample>
    ```json
    {
      "facebookAccountId": "uuid",
      "name": "Ad Account Principal",
      "externalId": "act_123456"
    }
    ```
    </RequestExample>

  </Accordion>

  <Accordion title="Listar Contas de Anúncios" icon="list">
    Lista as contas de anúncios com filtros e paginação.

    **Método:** GET
    **URL:** `/accounts/adaccounts`

    <ParamField query="page" type="number" default="1">
      Número da página.
    </ParamField>

    <ParamField query="limit" type="number" default="10">
      Itens por página.
    </ParamField>

    <ParamField query="facebookAccountId" type="string">
      Filtrar por conta do Facebook.
    </ParamField>

  </Accordion>

  <Accordion title="Adicionar Conta de Anúncios Diretamente" icon="link">
    Associa uma conta de anúncios existente diretamente a uma conta do Facebook.

    **Método:** POST
    **URL:** `/accounts/adaccounts/direct`

    <ParamField body="facebookAccountId" type="string" required>
      ID da conta do Facebook.
    </ParamField>

    <ParamField body="externalAdAccountId" type="string" required>
      ID externo da conta de anúncios.
    </ParamField>

    <RequestExample>
    ```json
    {
      "facebookAccountId": "uuid",
      "externalAdAccountId": "act_654321"
    }
    ```
    </RequestExample>

  </Accordion>

  <Accordion title="Remover Conta de Anúncios" icon="trash">
    Remove uma conta de anúncios do sistema.

    **Método:** DELETE
    **URL:** `/accounts/adaccounts/{id}`

    <ParamField path="id" type="string" required>
      ID da conta de anúncios.
    </ParamField>

    <Note>
      Retorna **204 No Content** em caso de sucesso.
    </Note>

  </Accordion>

</AccordionGroup>
